XML 相关的一点内容

2014-Apr-11, 20:11:01, Friday

想用 XML 来保存一点数据，因此在这边来作个笔记。

  • XML 所有元素都必须有关闭标签。

XML 声明不属于 XML 的部分，因此声明是没有关闭标签的。

  • XML 标签是大小写敏感的。
  • XML 标签必须正确嵌套。
  • XML文档必须有根元素。
  • XML 的属性必须加双引号。
  • < 等要写成&lt;。
  • XML 中的空格会被保留，而不是像 HTML 一样合并为一个。

XML 的命名应该遵循一些规则

  • 不能以标点或数字开头
  • 不能以xml/XML/Xml等开头
  • 不能包含空格
  • 尽量避免使用. - :
  • 可是使用非英语字母，但是不建议使用。

尽量使用元素，而要避免使用属性。属性仅仅用来描述与数据无关的信息，而用元素来处理信息。

如果 XML 文件有错误，程序就会停止执行。
